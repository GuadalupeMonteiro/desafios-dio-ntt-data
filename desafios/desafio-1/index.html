A tag p é utilizada para parágrafo.

A tag html envolve todo o conteúdo de semântica.

O head faz parte do html, contudo é a parte não visível. Ademais, na mesma se encontra as meta tags.

O body faz parte do html, é a parte visível do html.

Há tags que não possuem fechamento, é o caso do input. Mas ele precisa receber um atributo para que funcione. Como exemplo, pode ser citado: text, number, radio, color.

A tag img adiciona imagem. É relevante utilizar junto com o src para especificar o endereço onde se encontra a imagem, seja no computador ou web.Outro ponto importante é o alt para questões de acessibilidade.

O id é um identificador. Pode ser usado para adicionar estilos através do CSS, ou até mesmo, ser usado no Javascript. O id é um identificador único.

O class é utilizado para identificar elementos, contudo, pode ser atribuido a vários elementos.

O style adiciona um estilo CSS. Pode ser inline, interno ou externo.
